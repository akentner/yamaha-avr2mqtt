Command Details of SubUnit: NETRADIO
  • AVAIL
    [GET Command]
    @NETRADIO:AVAIL=?
    Retrieving the state of network link or the validity of NET RADIO operational status
    Initial Auto Feedback is Available
    [Parameters]
    Not Connected
    GET Only: indicating the network link is down.
    This status can be reported only via RS-232C connection since the network cable connection is not physically detected.
    Not Ready
    GET Only: indicating invalid state of operation
    Ready
    GET Only: indicating valid state of operation
    Back To Index

    @NETRADIO:PRESET=Parameter
    Operating Preset Bank (for USB/NET) state.
    [Parameters]
    Numerical Number: 1 to 40, Step: 1
    PUT Only: recalling specified number of preset bank.

  • MEM
    [PUT Command]

    [Parameters]
    Auto
    PUT Only: registering to a preset bank with automatically specifying the number.
    Numerical Number: 1 to 40, Step: 1
    PUT Only: registering to a preset bank of the number specified.
